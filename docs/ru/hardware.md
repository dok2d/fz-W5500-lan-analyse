# Оборудование и подключение

## Необходимое оборудование

- **Flipper Zero** с официальной прошивкой (OFW)
- **W5500 Ethernet-модуль** с SPI-интерфейсом и RJ45-разъёмом

Приложение протестировано с модулем **W5500 Lite**, разработанным для Flipper Zero, но подойдёт любая плата на чипе W5500 с доступом к SPI-пинам (WIZnet W5500-EVB, китайские breakout-платы и т.д.). Главное требование -- чип W5500 и выведенные пины MOSI, MISO, SCK, CS, RESET.

## Где купить

- [W5500 Ethernet Module for Flipper Zero](https://flipperaddons.com/product/w5500-ethernet/) -- готовый модуль с RJ45, подключается напрямую к GPIO-разъёму Flipper'а

## Схема подключения

```
Модуль W5500     GPIO Flipper Zero
─────────────    ─────────────────
MOSI (MO)    →    A7  (пин 2)
SCLK (SCK)   →    B3  (пин 5)
CS   (nSS)   →    A4  (пин 4)
MISO (MI)    →    A6  (пин 3)
RESET (RST)  →    C3  (пин 7)
3V3  (VCC)   →    3V3 (пин 9)
GND  (G)     →    GND (пин 8 или 11)
```

### Описание пинов

| Пин | Функция | Направление | Описание |
|-----|---------|-------------|----------|
| **MOSI (A7)** | SPI Master Out | Flipper → W5500 | Данные от Flipper к W5500 |
| **MISO (A6)** | SPI Master In | W5500 → Flipper | Данные от W5500 к Flipper |
| **SCK (B3)** | SPI Clock | Flipper → W5500 | Тактовый сигнал, генерируемый Flipper |
| **CS (A4)** | SPI Chip Select | Flipper → W5500 | Активный низкий; выбирает W5500 для обмена |
| **RESET (C3)** | Аппаратный сброс | Flipper → W5500 | Активный низкий; удерживается высоким, кратковременный импульс для сброса чипа |
| **3V3** | Питание | Flipper → W5500 | 3.3В от OTG-выхода Flipper |
| **GND** | Земля | Общий | Общая земля |

## Питание

Модуль W5500 питается от OTG 3.3В выхода Flipper Zero. Приложение автоматически включает OTG-питание при запуске и отключает при выходе. Внешний блок питания не нужен.

Типичное потребление тока -- 100-150 мА в активном режиме. Flipper может обеспечить это от батареи, но длительная работа разряжает батарею быстрее обычного.

## Ethernet-кабель

Используйте стандартный кабель Cat5/Cat5e/Cat6 с разъёмами RJ45. Подойдут и прямые, и кроссовые кабели -- современное оборудование поддерживает Auto-MDI/MDI-X.

Для прямого подключения к ПК (без свитча) подойдёт любой тип кабеля с современными сетевыми адаптерами.

## Совместимые платы W5500

Любая плата на чипе WIZnet W5500 будет работать при правильном подключении SPI-пинов:

- **W5500 Lite для Flipper Zero** -- plug-and-play, без пайки
- **WIZnet W5500-EVB** -- оценочная плата, нужны проводники
- **Китайские breakout-платы W5500** -- доступны на AliExpress и в магазинах электроники, нужны проводники

Приложение автоматически определяет чип W5500 при запуске, читая регистр версии. Если чип не найден, на экране появится "W5500 Not Found!".

## Советы по подключению

1. Дважды проверьте соединения перед включением -- неправильная SPI-разводка не повредит оборудование, но чип не будет обнаружен
2. Убедитесь в надёжном контакте на всех пинах -- нестабильные соединения вызывают ошибки SPI
3. Держите провода короткими (до 15 см) для надёжной SPI-связи на скорости
4. При использовании макетной платы убедитесь в чистых контактах без окисления
