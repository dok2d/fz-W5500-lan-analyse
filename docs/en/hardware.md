# Hardware Setup

## Required Hardware

- **Flipper Zero** running official firmware (OFW)
- **W5500 Ethernet module** with SPI interface and RJ45 jack

The app is tested with the **W5500 Lite** module designed for Flipper Zero, but any W5500-based board that exposes SPI pins will work (WIZnet W5500-EVB, generic breakout boards, etc.). The key requirement is that the board uses the W5500 chip and provides MOSI, MISO, SCK, CS, and RESET pins.

## Where to Buy

- [W5500 Ethernet Module for Flipper Zero](https://flipperaddons.com/product/w5500-ethernet/) -- ready-to-use module with RJ45, designed to plug directly into the Flipper's GPIO header

## Wiring Diagram

```
W5500 Module    Flipper Zero GPIO
─────────────   ─────────────────
MOSI (MO)   →   A7  (pin 2)
SCLK (SCK)  →   B3  (pin 5)
CS   (nSS)  →   A4  (pin 4)
MISO (MI)   →   A6  (pin 3)
RESET (RST) →   C3  (pin 7)
3V3  (VCC)  →   3V3 (pin 9)
GND  (G)    →   GND (pin 8 or 11)
```

### Pin Descriptions

| Pin | Function | Direction | Description |
|-----|----------|-----------|-------------|
| **MOSI (A7)** | SPI Master Out | Flipper → W5500 | Data from Flipper to W5500 |
| **MISO (A6)** | SPI Master In | W5500 → Flipper | Data from W5500 to Flipper |
| **SCK (B3)** | SPI Clock | Flipper → W5500 | Clock signal generated by Flipper |
| **CS (A4)** | SPI Chip Select | Flipper → W5500 | Active-low; selects W5500 for communication |
| **RESET (C3)** | Hardware Reset | Flipper → W5500 | Active-low; held high during normal operation, pulsed low to reset the chip |
| **3V3** | Power | Flipper → W5500 | 3.3V from Flipper's OTG power output |
| **GND** | Ground | Common | Shared ground reference |

## Power

The W5500 module is powered by the Flipper Zero's OTG 3.3V output. The app enables OTG power automatically on startup and disables it on exit. No external power supply is needed.

Current draw is typically 100-150 mA when active. The Flipper can sustain this from its battery, though extended use will drain the battery faster than normal.

## Ethernet Cable

Use a standard Cat5/Cat5e/Cat6 Ethernet cable with RJ45 connectors. Both straight-through and crossover cables work -- modern Ethernet equipment supports Auto-MDI/MDI-X.

For direct connection to a PC (without a switch), any cable type works with modern network adapters.

## Compatible W5500 Boards

Any board based on the WIZnet W5500 chip will work, provided you connect the correct SPI pins. Examples:

- **W5500 Lite for Flipper Zero** -- plug-and-play, no wiring needed
- **WIZnet W5500-EVB** -- evaluation board, needs jumper wires
- **Generic W5500 breakout boards** -- available from various electronics retailers, needs jumper wires

The app auto-detects the W5500 chip by reading its version register on startup. If the chip is not found, you will see "W5500 Not Found!" on screen.

## Connection Tips

1. Double-check pin connections before powering on -- incorrect SPI wiring will not damage hardware but the chip won't be detected
2. Ensure solid contact on all pins -- intermittent connections cause SPI errors
3. Keep wires short (under 15 cm) for reliable SPI communication at speed
4. If using a breadboard, ensure clean connections without corrosion
